## 🎯 ЧТО КАЖДЫЙ КЛАСС ДЕЛАЕТ

### App.java (главная программа)
```
Делает:
1. DbQuestionRepository.findAll()    → получить вопросы
2. ConsoleQuiz.askQuestion()         → выбрать случайный
3. Question.displayQuestion()        → вывести на экран
```

### DbQuestionRepository.java (работа с БД)
```
Методы:
- findAll()  → получить все вопросы из PostgreSQL
- insert()   → добавить новый вопрос в БД
```

### ConsoleQuiz.java (логика)
```
Методы:
- askQuestion()  → выбрать случайный вопрос
```

### Question.java (модель)
```
Поля:
- id              → уникальный ID
- questionText    → текст вопроса
- options         → варианты ответов
- correctOption   → индекс правильного ответа
- questionType    → тип вопроса

Методы:
- displayQuestion()  → вывести на экран
- getters/setters    → работа с полями
```

## 🔐 ВАЖНЫЕ ПРАВИЛА

1. **Всегда используй UTF-8 кодировку**
   - Консоль: `chcp 65001`
   - Java: `-Dfile.encoding=UTF-8`
   - БД: `?characterEncoding=UTF-8`

2. **Используй PreparedStatement для БД**
   - ✅ Правильно: `ps.setString(1, value)`
   - ❌ Неправильно: `sql = "... '" + value + "'"`

3. **Используй try-with-resources для ресурсов**
   - ✅ Правильно: `try (Connection conn = getConnection())`
   - ❌ Неправильно: Забыть `conn.close()`
